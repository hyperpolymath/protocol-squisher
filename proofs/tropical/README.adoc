== What this directory contains

Formal foundations for the claim that Protocol Squisher's 4-class transport
semilattice (Concorde / Business / Economy / Wheelbarrow) is a *tropical
optimisation problem* — specifically, that optimal adapter path search is
Dijkstra on an idempotent (tropical) semiring.

== The mathematical claim

Protocol Squisher classifies adapter paths by their *bottleneck quality*:
the quality of the worst step in the path.  Finding the optimal path
(maximum fidelity) is equivalent to minimising the bottleneck cost, which
is the classical *minimax path problem*.

The minimax path problem is solved exactly by Dijkstra's algorithm on the
*(Nat, min, max, ∞, 0)* semiring, also called the **min-max semiring** or
**bottleneck semiring**.  This semiring is:

* Idempotent (min(n,n) = n) — making it a *dioid*
* A tropical semiring in the general sense (idempotent addition)
* Dual to the max-plus semiring via the map g ↦ (maxGrade − g)

== Relationship to `tropical-resource-typing`

The `tropical-resource-typing` repository proves a max-plus CommSemiring
instance (⊕ = max, ⊗ = +) and uses it to grade session types.  Both
structures are dioids; they are related by an order-reversing isomorphism
on [0, maxGrade].

The analogue of `tropical_grade_le_sequentialTotal` (proved there) holds
here: the bottleneck cost of a path is always ≤ the naive sequential sum
of step costs (because max(a,b) ≤ a+b for naturals).  This is proved in
`pathCost_le_sequential`.

== What is proved (zero sorry)

In addition to the semiring axioms and duality theorems, the following path
optimality results are now fully proved:

[cols="2,3"]
|===
| Theorem | Content

| `foldl_tcAdd_le_init`
| `foldl min acc xs ≤ acc` — the running minimum never exceeds the starting bound.

| `foldl_tcAdd_le_mem`
| `foldl min tcZero xs ≤ pathCost Q` for all `Q ∈ xs` — global minimum property.

| `minimax_path_optimal`
| If `pathCost P = foldl min tcZero alts`, then `P` is optimal (≤ every alternative).
  Full proof using the two helper lemmas; no `omega` placeholder.
|===

== Open items

. **Full graph-algorithmic Dijkstra** — the `minimax_path_optimal` theorem
  proves optimality for the *foldl-min* model (linear list of alternatives).
  Generalising to Dijkstra on an arbitrary weighted graph (Mohri 2002) requires
  a graph library or further formalisation effort.
. **Explicit semiring homomorphism** — the order-reversing map from the min-max
  semiring to the max-plus semiring should be proved as a semiring homomorphism,
  not just as a lattice anti-isomorphism.
. **Connection to `TropicalSessionTypes.lean`** — construct an explicit functor
  between the CommSemiring instances.
